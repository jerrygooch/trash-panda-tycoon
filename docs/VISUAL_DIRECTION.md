# Visual Direction — Trash Panda Tycoon

## Style Summary
Cute trash raccoon mobile game. Bold, readable, phone-friendly shapes. High contrast, playful but not cluttered. No AI-generated text in images.

## Core Palette
| Role | Color | Hex |
|------|-------|-----|
| Background | Dark navy/purple | `#14141E` |
| Accent / Coins | Warm gold | `#FFD700` |
| Food bin | Green | `#66BB6A` |
| Metal bin | Gray/silver | `#78909C` |
| Plastic bin | Blue | `#42A5F5` |
| Mess / Danger | Red/orange | `#FF5252` |
| Text primary | White/off-white | `#FFFFFF` / `#E0E0E0` |
| Panel surface | Dark slate | `#1E1E2A` |

## UI Principles
- Large touch targets (min 44px per Material Design)
- Category labels are Godot text, not baked into images
- Emoji fallbacks alongside any custom icon textures
- Rounded corners on all panels (radius 8–12px)
- Consistent spacing (4px grid)

## Asset Guidelines (for generation)
- Transparent PNG background
- No text, watermark, logo, or signature in images
- Readable at 60x60px for items, 200px for bins, 400px for mascot
- Clean outlines, high contrast against dark backgrounds
- Cute/simplified proportions, not photorealistic

## Typography
- In-game labels: Godot default font, size 14–28 depending on hierarchy
- Button text: size 20–24
- HUD text: size 16–22
- Debug/tiny: size 10–12

## Tone
Playful, mischievous raccoon running a trash empire. "Stupid, addictive, mobile." Not serious.
